Hexagon Auto Updater

SourceCode

Description

The HexagonAutoUpdater is a simple AutoUpdater written in Java,
it is based on HTTP and is only able to download updates from HTTP servers

Howto Use

Inorder for it to function it requires the path of the local version file
and the URL of the Webserver which hosts the update files, the version file must only contain a version of this format: X.X
if the server-side version is greater than the clients version, it will attempt to download a file named "files.txt" from the server.
This file contains a list of files to download (directories must be compressed). It will then download all these and place them in the
same directory as the downloader is located in.
If you have any other questions, you can contact the authors.
